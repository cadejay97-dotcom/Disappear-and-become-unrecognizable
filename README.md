在这个时代，从诞生想法，进行研究，到投入开发，到验证成败的速度是以往所不具备的。（学会撰写提示词、让AI调研）成败往往一个月就能验证，转向也很快，所以我之前作为非编程人士对这种商业化途径感兴趣。这是后话了。
我的<strong>初衷是“情绪价值”</strong>。没错，我把前端的设计看作“审美”“设计”，通过前端的语言我们得以自由设计动效让人耳目一新，交互让人“爽”到起飞的网站，界面。
所以我眼里的前端，最开始是一个设计工具，给我们带来“情绪价值”。
<strong>前端等同于=音乐、艺术</strong>
