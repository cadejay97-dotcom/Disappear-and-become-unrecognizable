# Disappear-and-become-unrecognizable
这份文件将收录我的前端学习过程中编写的代码，同时记录我在这一个阶段（2026）所坚持的七个重要的原则。
在这里我记录下我学习前端的初衷，时时回顾时时温习。
在2025年AI编程的效果可以说让不懂代码的人拥有一个“程序员”
Cursor、Trae、Codex、Lovable、Bolt、Antigravity、Replict等等vibe coding应用层出不穷。
我只看事实：1、在过去的时间里，非编程人员通过vibe coding构建自己的app上线这条路至少被验证可行；
2、Gemini前端的能力对基础前端的效率提升显著；3、中小程序员团队开发海外SaaS路径可行。
这些不是我学习前端的初衷，我的初衷是“情绪价值”。没错，我把前端的设计看作“审美”“设计”，通过前端的语言我们得以自由设计动效让人耳目一新，交互让人“爽”到起飞的网站，界面。
所以我眼里的前端，最开始是一个设计工具，给我们带来“情绪价值”。
前端等同于=音乐、艺术
